# GodotSTG
**NOTE:** *this plugin is still in development*
> A Godot plugin that offers an easy way to make STG battles/games

GodotSTG is a series of state machines that offers you a streamlined way to make bullet hell/shoot 'em up games. It aims to be powerful enough to create complex patterns while being simple enough to get into easily. It aims to offer:
- [x] A way to create and configure battles with very little coding
- [x] Sequenced spawning to change spellcards on the fly
- [ ] Tons of options to make anything your heart desires
- [x] A way to add your custom bullets
- [x] Performance improvements that allow thousands of bullets with little overhead
- [ ] A way to play individual spells
- [x] Tons of signals to adapt the plugin into any workflow
- [x] Manupulate player and enemy positions for each spellcard
